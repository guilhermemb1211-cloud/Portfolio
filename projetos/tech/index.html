scraping_noticias.ipynb
BeautifulSoup requests

Automação de coleta de dados jornalísticos do portal Estadão usando técnicas de web scraping. O script realiza requisições HTTP, faz o parsing do HTML com BeautifulSoup e extrai os títulos de notícias em tempo real — demonstrando como obter dados não estruturados da web para alimentar pipelines de análise.

Python
import requests from bs4 import BeautifulSoup # Requisição ao portal de notícias url = 'https://www.estadao.com.br/' response = requests.get(url) soup = BeautifulSoup(response.text, 'html.parser') # Extração de manchetes for headline in soup.find_all('h2', class_='headline'): print(headline.get_text().strip())

Análise de Sentimentos com SentiLex
sentilex_roteiro_va_01.ipynb
NLP SentiLex

Processamento de Linguagem Natural (NLP) em português com o léxico de sentimentos SentiLex-PT. O notebook implementa um pipeline de análise de sentimentos para classificar textos em positivo, negativo ou neutro — técnica aplicável em monitoramento de redes sociais, análise de reviews de produtos e feedback de clientes.

Python – Pipeline NLP
# Carregamento do léxico SentiLex-PT sentilex = {} with open('SentiLex-lem-PT02.txt') as f: for line in f: word, polarity = parse_entry(line) sentilex[word] = polarity # +1 positivo / -1 negativo # Classificação de sentimento por texto def classify_sentiment(text): tokens = text.lower().split() score = sum(sentilex.get(t, 0) for t in tokens) return 'positivo' if score > 0 else 'negativo' if score < 0 else 'neutro'
