- **Formação** – Engenharia Mecatrônica (Mackenzie) + Intercâmbio (Pittsburg State University)

## Tecnologias

- HTML5 / CSS3 (sem frameworks)
- JavaScript vanilla (scroll animations)

## Como visualizar

Abra o arquivo `index.html` diretamente no navegador ou sirva com qualquer servidor estático:

```bash
python -m http.server 8000
```

Acesse `http://localhost:8000`.
